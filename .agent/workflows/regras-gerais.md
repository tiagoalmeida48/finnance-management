# Regras Gerais

## Idioma
- Sempre responder em **Português (Brasil)**
- Usar termos técnicos em inglês quando apropriado (ex: "commit", "build", "deploy")

## Gerenciador de Pacotes
- Sempre usar **pnpm** ao invés de npm ou yarn
- Exemplos:
  - `pnpm install` ao invés de `npm install`
  - `pnpm run dev` ao invés de `npm run dev`
  - `pnpm run build` ao invés de `npm run build`
  - `pnpm add <package>` ao invés de `npm install <package>`
